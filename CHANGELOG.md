# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-07

### Added
- Initial release
- Multi-site network support with subsite-specific settings
- Secure Custom Field (SCF) options management
- Runtime rules engine for access control and validation
- Admin interface for subsite configuration
- Composer support for WordPress MU-plugin installation
- Comprehensive test suite with PHPUnit
- Code standards enforcement with PHP_CodeSniffer
- Complete documentation and API reference
- Filter and action hooks for extensibility
- Support for three rule types: access_control, field_validation, content_restriction
